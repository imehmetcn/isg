# İş Güvenliği Yönetim Sistemi

İş güvenliği uzmanları için geliştirilmiş kapsamlı bir yönetim sistemi.

## Özellikler

- Risk Değerlendirme
- Olay Takibi
- Eğitim Yönetimi
- Denetim ve Kontrol
- Acil Durum Planı

## Teknolojiler

- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- NextAuth.js

## Geliştirme

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Production build
npm run build
```
